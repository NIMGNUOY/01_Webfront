영역을 나누는 형식(성질)

            
Block 형식

- 공간을 수직 분할
- 크기를 지정하는 width, height 속성을 사용할 수 있음

ex) p tag / pre tag (문단 나누기), h1~h6 (제목 영역 나누기),
    hr(수평선 넣어서 화면 나누기), div(영역나누기)

        

            
Inline 형식

- 공간을 수평 분할
- 크기를 지정하는 width, height 속성을 사용 할 수 없음

ex) strong/b, em/i, mark, span(한 줄을 나누는 용도)

block(div) / inline(span) 차이점 1 : 줄바꿈(개행)

뒤에 오는 내용은 다음줄에 작성이 된다.
