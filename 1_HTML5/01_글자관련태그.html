글자 관련 태그

b 태그 : 단순히 글자를 굵게 표시하는 태그
i 태그 : 단순히 글자를 기울이는 태그
b,i 태그 중복 (태그는 중복 적용이 가능하다)


strong tag : 중요성, 심각함 등을 표현하는 강조 태그
em tag : 문장 일부분의 중요성을 강조할 때 사용

mark tag : 형광펜 효과를 나타내는 태그
u tag : under line(밑줄)을 나타내는 태그
s tag : 취소선(잘못된 글을 표시할 때)
del tag : 삭제선(없어진 텍스트 표시)
small tag : 작은 글자 표시
sup tag(윗첨자) : x2
sub tag(아래첨자) : log10
abbr tag : 마우스 오버 시 툴팁 출력
HTTP

태그들은 해당 태그 내에서 중첩으로 사용 가능하다.
굵은글자를 사용할 수도 있고, 기울이거나 취소선을 넣는 등
다양하게 사용할 수 있다.
