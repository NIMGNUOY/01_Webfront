ol 태그

* ol tag의 type 속성
type="a" -> 소문자 알파벳
type="A" -> 대문자 알파벳
type="i" -> 소문자 로마숫자
type="I" -> 대문자 로마숫자
type="1" -> 숫자(기본값)
start 속성 : 시작 번호 지정
reversed 속성 : 순서 반대로
